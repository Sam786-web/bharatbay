# 🌾 BharatBay - E-Commerce Platform for Farmers

## 📌 Overview
BharatBay is an e-commerce platform designed to connect farmers directly with consumers. The platform eliminates unnecessary middlemen and enables farmers to sell their products efficiently while providing customers with fresh agricultural products at fair prices.

## 🚀 Features
- Farmer Registration & Login
- Customer Registration & Login
- Product Listing & Management
- Product Search & Browsing
- Shopping Cart
- Order Management
- Responsive User Interface
- Secure Authentication

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Sam786-web/bharatbay.git
```

2. Navigate to project directory

```bash
cd bharatbay
```

3. Install dependencies

```bash
mvn clean install
```

4. Run the application

```bash
mvn spring-boot:run
```
